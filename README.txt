Kurze Erläuterung zu den Dateien im Projektordner:

- save.txt:	Beinhaltet den aktuellen Spielstand, d.h. das höchste geschaffte Level. Nur dieses und niedrigere Levels können gespielt werden. Im Abgabezustand ist dort 304 (Zone 3, Level 4), es können also
		alle Levels gespielt werden. Um einen "normalen" Spieldurchlauf zu starten, einfach save.txt löschen.
